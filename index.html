Graphic Design Styles

- A Visual Guide

pop art

  • uses vibrant, contrasting hues to grab attention
  • draws from tv, ads, comics, everyday objects
  • employs humor critique contemporary culture
  • features clear, striking lines and shapes

swiss style

  • employs strict grids for organized content placement
  • draws from tv, ads, comics, everyday objects
  • features straightforward, unembellished photos
  • uses asymmetry for dynamic visual interest

flat design

  • utilizes grids and open space for a cohesive layout
  • design elements have a clear purpose
  • features basic elements, clear typography, and flat colors
  • uses contrasting colors for engaging visuals

minimalism

  • focuses on essential elements only
  • emphasizes negative space to highlight the main
  • uses minimal colors to maintain focus
  • bold, straightforward fonts enhance readability

modernism

  • clean, legible typefaces emphasizing readability
  • abstract designs using circles, squares, and triangles
  • structured layouts for clarity and order
  • bold combinations for visual impact and attention

post-modernism

  • mixes various type styles, sizes, and orientations
  • introduces humor and surprise, challenging conventional narratives
  • blends different styles and historical forms

contemporary

  • features strong, experimental fonts
  • reflects movements towards sustainability and inclusivity
  • combines digital, hand-drawn elements for a versatile visual
  • uses bold, unconventional colors to stand out

art deco

  • characterized by bold triangles, rectangles, and zigzags
  • features deep, vibrant hues
  • incorporates glossy finishes and metallics
  • focuses on balanced, harmonious compositions

brutalism

  • utilizes stark contrasts bold color flashes
  • draws from tv, ads, comics, everyday objects
  • showcases the “bones” of the structure
  • features makeshift fonts and unconventional typefaces

psychedelic

  • features highly saturated, contrasting colors for a strike
  • includes complex shapes for hallucinogenic effects
  • dream-like elements to evoke consciousness
  • uses unique, stylized, often distorted hand-drawn text

bauhaus

  • geometric shapes
  • clean lines
  • restricted color palette
  • primary colors for contrast

scrapbooking

  • combines various materials and embellishments to create a rich, tactile experience
  • emphasizes personal touches and memorabilia
  • incorporates decorative element
